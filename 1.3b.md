## 1. Permitir acceso a la interfaz web desde cualquier ordenador

Editar `/etc/cups/cupsd.conf`:

```
<Location />
  Order allow,deny
  Allow @LOCAL
  Allow all
</Location>

<Location /admin>
  Order allow,deny
  Allow @LOCAL
  Allow all
</Location>
```

Reiniciar CUPS:

```
sudo systemctl restart cups
```

## 2. Habilitar interfaz web solo en local o desde una IP concreta

Solo local:

```
<Location />
  Allow localhost
  Deny all
</Location>
```

Permitir solo desde una IP (ej: 192.168.56.11):

```
<Location />
  Allow 192.168.56.11
  Deny all
</Location>
```

## 3. Control de acceso por usuarios y grupos

```
<Location /admin>
  AuthType Default
  Require user pepet
</Location>

<Location /admin/jobs>
  AuthType Default
  Require group profesores
</Location>

<Location /admin/printers>
  AuthType Default
  Require valid-user
</Location>
```

Reiniciar CUPS:

```
sudo systemctl restart cups
```
