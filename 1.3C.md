# Actividad de Configuración de Impresión

## 1. Imprimir desde consola un archivo de imagen en papel A5 apaisado
usando `lp`:
```
lp -o media=A5 -o landscape ruta/archivo.png
```

## 2. Configuración predeterminada para todos los usuarios
Establecer opciones por defecto para tamaño A4 y margen izquierdo de una pulgada:
```
lpoptions -o media=A4 -o page-left=72
```

## 3. Configuración para todos los clientes de red
Ajustar la resolución predeterminada a 150 dpi:
```
lpadmin -p PDF -o resolution=150dpi
```
